Form Demo

This is a quick form demo. Currenly not using many features of the FormModule but is the basic setup of both types of form usage in Angular. You can find the source code on GitHub open_in_new

Model Form

This is a clasic form in Angular. The form is controlled by bindings in the template linking to a component. Taking advantage of the two-way data binding provided by Angular

Reactive Form

This is a more advanced way to build forms in Angular. Which allows you to dynamically control the form and form validators
